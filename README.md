# Proyecto Cisled – Sitio Web Institucional

## Descripción
Sitio web institucional desarrollado para el emprendimiento Cisled, orientado a la presentación de servicios de soporte y soluciones IT para usuarios finales y pequeñas oficinas.

## Objetivo
Contar con una presencia web simple y clara que permita:
- Comunicar servicios ofrecidos
- Facilitar el contacto con clientes
- Brindar información básica del emprendimiento

## Alcance funcional
- Página informativa de servicios
- Sección de contacto
- Contenido institucional
- Galería visual de trabajos realizados

## Rol desempeñado
- Relevamiento de requerimientos del negocio
- Definición de funcionalidades del sitio
- Organización del contenido y estructura
- Implementación técnica básica del sitio

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Observaciones
El foco del proyecto estuvo en el análisis de necesidades del negocio y la definición de funcionalidades, priorizando claridad, simplicidad y facilidad de uso.
